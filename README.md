# Calculator
It takes a little skill to create a calculator with basic mathematical functionalities using CSS, HTML, and JavaScript. Calculators require an interactive interface to perform basic functions such as addition, subtraction, division, and multiplication.
# Scientific Calculator App
# Overview
This is a fully functional scientific calculator web application built with HTML, CSS, and vanilla JavaScript. It features a dark-themed interface with gradient backgrounds, supports basic arithmetic operations (+, -, ×, ÷), decimal points, and keyboard input. The calculator displays previous and current operands with smooth animations and responsive design.
​

# Features
Dual Display: Shows previous operation + current input with proper formatting (thousands separators).

Full Keyboard Support: Numbers, operators, Enter (=), Backspace (DEL), Escape (AC).

Operation States: Visual feedback with active operator highlighting and hover effects.

Smart Input: Prevents multiple decimals, handles zero replacement, floating-point precision.

Responsive Design: Adapts to mobile screens with adjusted padding and font sizes.

Animations: Fade-in entrance, button scaling on hover/active states.
​

# Operations Supported
Button	Function	Keyboard Shortcut
AC	Clear All	Escape/C 
​
DEL	Delete Last Digit	Backspace 
​
+, -, ×, ÷	Arithmetic Operations	Keys +, -, *, / 
​
=	Compute Result	Enter/= 
​
0-9, .	Number Input	Number keys 
​
Division by zero protection with alert and clear.
​

# Usage
Save as task1.html and open in any modern browser.

Click buttons or use keyboard for calculations.

Enter numbers → operator → numbers → Enter for results.

Works offline with no external dependencies.
​

# Technologies
HTML5: Semantic structure with grid-based button layout.

CSS3: CSS Grid (4-column buttons), Flexbox display, gradients, custom shadows, toLocaleString() formatting, media queries (480px breakpoint).

JavaScript: Class-based architecture, event delegation, parseFloat() parsing, dynamic DOM updates, keyboard event listeners.
